# 简介

本工程为一个USB烧录器，用于SPI Flash（主要支持w25q80及兼容器件）和I2C EEPROM(主要为AT24C64及兼容器件)的烧录。主控为STM32。

## 目录简介

* Doc:设计文档及笔记
* HW:用于验证固件的电路板的设计。
* FW:固件源代码
* SOFT:简易上位机源代码（只支持SPI Flash和I2C EEPROM的读取）
